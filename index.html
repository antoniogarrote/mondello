Manage Docker Machines

Create, run, stop and destroy Docker machines with a few clicks. Explore different building options for the machines and track the state of your machines from a simple user interface.

Pull Index Images

Search images in remote indices. Pull images and synchronize remote changes.

Track Container Logs

Aggregate logs from different running containers. Use the provided search interface to find relevant information from the graphica user interface.

Run Containers

Start containers from Mondello's user interface into an regular terminal to interact with the process being executed. Track the whole life-cycle of a container: execution, termination. Attach to running containers with the click of a button.

Manage Index Credentials

Keep the list of the index services you're logged-in. Add credentials for new services and remove existing ones.

Search Meta-Data

Use the search interface to filter the list of images and containers with the meta-data information available in the different services.
